# Snakeloader AI Operations

This folder is the operating source of truth for Claude/Codex work in this repo.

- `AI_OPERATING_CONTRACT.md` - authority, gates, RCA discipline, and stop conditions.
- `TASK_PACKET_TEMPLATE.md` - required starting packet for non-trivial work.
- `REVIEW_VERDICT_PROTOCOL.md` - Codex/Claude review vocabulary and blocker standard.
- `CLI_AGENT_RUNBOOK.md` - native Claude/Codex invocation patterns for this machine.

These files are intentionally small and explicit. Do not replace them with external harnesses or plugins without audit, rollback plan, and explicit approval.
