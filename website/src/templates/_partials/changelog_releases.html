v1.3.7 April 27, 2026 Stability

Reliability & Crash-Free Update

  • Crash fix: home page no longer crashes when rendering imported library thumbnails — resolves the v1.6.2/v1.6.3 "No host specified in URI" crash-loop.
  • Crash fix: tap-to-play / pause no longer crashes when the player is closing.
  • Windows + Linux first-launch reliability: gallery-dl now downloads from a daily-built mirror with a fallback chain so an upstream-empty release cannot brick fresh installs.
  • Image download is now optional at first launch — yt-dlp + ffmpeg succeed alone if a gallery-dl mirror is unreachable; users still reach the home screen.
  • Telemetry: update banner show, click, and dismiss events are now recorded so adoption stalls are diagnosable instead of guessed.
  • Backend release traceability: /health and /version now expose real build SHA and version.
  • CI workflow: Linux artifacts are now correctly attached to GitHub releases (artifact-symmetry gate added).
  • Pre-release validation: new scripts/preflight_release.sh checks workflow YAML symmetry, all binary URLs return 200, and backend release records are healthy before tagging.
v1.3.6 April 25, 2026 Stability

Stability & Hardening Update

  • Cinematic Aurora UI refresh across top nav, browser, home, premium, support, and YouTube surfaces.
  • Auto-update reliability: streaming SHA-256 verify with HTTP Range resume so an interrupted update continues from the last byte.
  • Supply-chain integrity: yt-dlp installs verify against upstream SHA2-256SUMS manifest before accepting the binary.
  • Windows installer signing path validated end-to-end (Sectigo HTTPS timestamp, RSA-baseline gate); macOS entitlements tightened (no JIT, no unsigned executable memory).
  • VidCombo migration: license-key import is now idempotent under antivirus interference; legacy folder scan covers Dropbox + Google Drive paths.
  • Notification permission channel: null-guarded against pre-init calls to fix MissingPluginException on cold start.
  • Keychain probe: probe-once strategy with 24h fallback cache eliminates per-launch retry spam on unsigned debug builds.
  • Observability: 5 critical Sentry breadcrumbs for marker-policy, supply-chain hash, cache hit/miss, auto-update resume, secure-credential fallback.
